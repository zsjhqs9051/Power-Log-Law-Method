Power-Log Law Method Code

Introduction
	convert the 2D shear stress distribution from SRH-2D to 2.5D shear stress distribution.
	1. convert the shear stress contour of the riverbed to 2.5D shear contour
	2. convert the shear stress profile of the chosen cross-section line to 2.5D shear profile and calculate the average value

Required Input:
	1. roughness height in meter
	2. 2D data
		Data order: Coordinate_1, Coordinate_2,U_ave Depth
		Coordinate_1 and Coordinate_2:
		   Surface Field: X,Y
		   Cross-Section: distance,elevation
